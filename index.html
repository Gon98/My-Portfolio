Redflix

Desenvolvimento web

Desenvolvido na licenciatura para o projeto final de multimédia. A Redflix é uma plataforma de streaming que permite o upload de vídeos, avaliações, comentários e a categorização de filmes. Todo o conteúdo do site é gerido, através de um painel de admin, que só tem acesso quem tiver as permissões de administrador.

  • Framework: Django
  • Linguagem: Python
  • Base de Dados: SQLite

Estúdio de Tatuagens

Desenvolvimento web

Rock Tattoo Studio foi desenvolvido no âmbito do projeto final de CTeSP. É um e-commerce que fornece produtos ligados ao mundo das tatuagens, quer sejam clientes, ou a outras entidades. O site é constituído por uma página informativa sobre o estúdio, uma loja e um serviço de marcações. Conta ainda com um backoffice para fazer todo o gerenciamento do site.

  • Framework: Bootstrap
  • Linguagens: HTML/CSS/JS/PHP
  • Base de Dados: MySQL
